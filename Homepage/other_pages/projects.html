CSEA Website
Summer 2017
It involved developing an extensive website for the Computer Science and Engineering Association, IIT Bombay to be used as a platform for notifying the concerned regarding updates, upcoming events, photographs, council members and achievements.
UnBlock Me Solver
Summer 2017
I developed a solver for the famous Unblock Me puzzle using Java SE 8 and Swing GUI. I implemented a brute force algorithm that gives a graphical, shortest, step-wise solution to the puzzle; which can be given as input either through the terminal or the GUI.
Tank Wars - The Game
Spring 2017
I developed an enjoyable, single-player tank-battle game using Racket with the objective of the game being destroying all enemies before they destroy us. The gameplay was simple, with 3 types each of weapons and enemies.
Desktop Buddy
Spring 2017
This project was developed in Python by a group of five people for the HACK-U Yahoo! hackathon at IIT Bombay. It is basically a speech-cum-text based desktop manager that fetches news, cricket scores, weather forecast, downloads and plays songs and even changes the desktop background on giving a vocal or written command.
